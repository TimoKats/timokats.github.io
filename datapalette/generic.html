DataPalette docs


URL scraping (first tab)

Methods

There are three methods in DataPalette to collect URLs with. The first method is through collecting all the links on a page, the second method is through collecting all the links on a page containing a specific substring, and the third method is through submitting a custom list of URLs. To use one of these methods simply select one of them in the first section of the URL scraper. After doing this you can move on to the second step of submitting URL(s).

Submitting URLs

In the second box it's possible to submit URL(s) and an optional substring. What you have to fill in here depends on the method of choice. If you've selected all links then simply fill in the URL box and DataPalette will collect all the links on that page. If you've selected all links containing a substring then fill in the URL and the substring in the optional field. Finally, if you've selected custom then fill in all URLs you'd like to scrape (seperated with a space) in the URL box and leave the optional field empty.

Export

After filling in the form click on Export url file and complete the file dialog.

Result and making URL files without using the scraper

The URL file that is created is simply a txt file with a URL on every newline. Thus, it's also possible to create your own url file (through a script or download) in this format and use it in DataPalette. This product is meant to be felixible so compatibillity shouldn't be an issue.


Content scraping (second tab)

Open URL file

By clicking on the button Open url file you can select the file you just made in the URL scraper. To reiterate, you can also select a file you didn't make through the URL scraper. As long as it's a txt file that contains a URL file on every line.

Explore URL file

After opening a URL file this section should show the meta properties that can be scraped through the dropdown menu. To start exploring simply select one of these items in the dropdown menu and view a sample in text or html with the button see text only or see html-code below.

Add items to scraper

Next, adding items to the scraper. To do this, simply click the button add item to scraper when the sample in the exploration window fits your needs. Please note that, only if an item is loaded in the exploration window it can be added to the scraper.

Export items to CSV

After filling in the form click on Export as CSV to export your items to csv for every URL in the URL file. Please keep in mind that this can take a while. So if it looks like the program is not responding it's best to wait. Once the program is complete it will show a notice with the amount of succesfully exported rows.


Other

Tools

In the menu bar on the top of the screen there are two different tools. The first tool is to see the selected items (that belong to the scraper) and the second tool is to see the URLs loaded by the URL file.
