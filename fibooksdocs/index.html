class balance_sheet.balance_sheet(company_name)

Bases: object

This class contains the balance sheet. The important objects are: content (contains the data), template (contains the excel template), standard_categories (contains categories like ‘current assets’ and ‘current liabillity’) and custom_categories (contains categories that are added by the user)

add_account(name, value, category=None)

Adds an account to content. The category is an optional parameter. For example, you can use this function to add ‘cash $20’ in ‘current assets’.

add_custom_category(name, accounts)

Adds a custom category along with the accounts that belong to that category. For example, you can use this function to define ‘cash equivalents’.
